# Extract-Load-Transform of your last 50 days in Spotify

## First of all you've to setup these variables in a `.env` file:
- `CLIENT_ID`
- `CLIENT_SECRET`
- `SPOTIPY_REDIRECT_URL`
- `DATABASE_LOCATION`

### Technologies used on this project:
- Python
- SQL
